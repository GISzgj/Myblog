HTML

概念

超文本标记语言，可以展示文字，图片和音视频等。主要用于描述一个页面，是页面的骨架。

操作思想

标签类似为一个容器，可以将数据放入容器内。页面中的数据，也能通过各种标签的属性值进行简单修饰。

特点

语法简单；较为宽松

标签固定，每个标签都有特定的语义

无序列表

不区分顺序的列表

  • ul 和 li 组成
  • ul：表示的是无序列表的整体，用于包裹li标签（只能包含li标签）
  • li： 用于表示无序列表的每一项，用于包含任何内容（任意内容）

有序列表

区分顺序的列表

  • 由ol和li组成

  • ul：表示的是有列表的整体，用于包裹li标签（只能包含li标签）

  • li：用于表示有序列表的每一项，用于包含任何内容（任意内容）

    类似上边无序列表的代码

自定义列表

在网页底部导航中通常使用自定义标签表示

  • dl:表示自定义列表的整体，里面包含dt和dd
  • dt:表示自定义列表的主题（标题）（可包含任意内容）
  • dd:表示自定义列表中的每一项的说明（可包含任意内容）

链接标签

超链接

href属性

  • 外部链接 <a href="https://www.baidu.com">百度</a>
  • 内部链接 <a href="./as.html">超链接</a>属性值常用的是相对路径

target属性

  • <a href="#">空链接</a>
  • 可以实现回到顶部的功能
  • 在项目中不确定最终跳转的页面

表格标签

  • table标签：表示表格的整体

  • tr标签：表示表格的每一行

  • th标签：表示 表头单元格

  • td标签：表示单元格

  • caption标签：表格大标题，表格的整体标题，默认居中

  • 嵌套关系 table>tr>td = th

    合并单元格

    单元格合并遵循左上原则

    • 跨行合并：上下合并只保留最上

      <td rowspan="2">111</td> 上下合并两格。

      rowsapan="(单元格合并的个数包含自身) "

    • 跨列合并：左右合并只保留最左边的

      <td colspan="2">111</td> 左右合并两格

      colsapn"(单元格合并的个数)"

input 其他属性

文本框下的两个重要属性：

其他属性

  1. placeholder

    属性值placeholder=““占位符，用于提示用户输入信息

    昵称：<input type="text" placeholder="请输入昵称"/>

  2. name

    （常用单选框）分组：有相同name的被分为一组

  3. checked

    默认选中：直接将要默认的后边加上这个属性

  4. multiple

    多用于文件选择框：多文件选择

若要实现按钮的功能，需要配合form标签，将input的内容放到form下
